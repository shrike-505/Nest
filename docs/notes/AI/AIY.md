### 启发式搜索

- 定义启发函数$h(n)$，用于评估节点$n$到目标节点的 path 的最小代价。
- 定义评价函数$f(n)$，用于选择节点$n$的下一个节点。

!!! example "贪婪最佳优先搜索"
    - $f(n) = h(n)$

!!! example "A*搜索"
    - $f(n) = g(n) + h(n)$
    - $g(n)$ 为从初始节点到节点$n$的实际代价。（当前最小代价）

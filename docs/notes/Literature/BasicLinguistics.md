# 语言学基础

> 任课教师：彭利贞

!!! note "Link"
    <https://socralin.github.io/2025/02/12/Basic-Linguistics-Learning/>

!!! definition "语言学是什么"
    - 语音
    - 词汇
    - 语法
    - 语义
    - 语用
    - 文字（副产品）

!!! definition "什么是语言？"
    - 结构：语言是一种符号系统
        - 符号：用 A 表示 B，A 和 B 没有必然的联系（皮尔斯：图像、索引、象征）
    - 功能：语言是人类最重要的交际工具，是人类最重要的认知、思维工具
    - 基本符号：词
        - 发音+语义=符号<->现实中的对象
        - 语言=词＋规则

**口语**是语言的原始形式，**书面语**是在口语基础上产生并发展的。
